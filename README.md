# Selina's Way to Success — Job Application Tracker

A single-page job application tracker: log applications, track pipeline status (Saved → Applied → OT → VI → PI → Superday → Offer), manage linked to-dos, and see a dashboard with a status donut, pipeline funnel, applications-over-time chart, and milestone badges.

All data is stored locally in your browser (`localStorage`) — nothing is sent to a server, and data is private to whichever browser/device you use it on.

## Live site
Once GitHub Pages is enabled for this repo, the app is available at:
`https://<your-username>.github.io/<repo-name>/`

## Local use
Just open `index.html` in any browser — no build step, no dependencies.

## Data & privacy
- Data lives in your browser's local storage only.
- Use the **Export backup** button in the app to download a JSON copy anytime.
- Clearing your browser data (or switching browsers/devices) will not carry your data over — export first if you need a backup.
